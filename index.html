NType

NType is a 4D communication tool.

Key Controls

  • Backspace Removes the last character on screen
  • A–Z Types a letter in 4D extrusion
  • Space Pause/play the animation
  • Scroll While Paused Scrub through the animation

Actions

  • Share Message Get a URL to share the current text and rotation parameters
  • Download OTF Download an OTF of (maybe) the current rotation. This part only works in chrome and doesn't work great yet.
